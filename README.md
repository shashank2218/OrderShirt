# OrderShirt
This project is for ordering a T Shirt from a web application

### Tools required to run the project
1. Intellij idea or Eclipse
2. Maven build tool

### Steps to run test scenarios
##### From Intellij
1. Go to TestRunner class.
2. Edit tags.
3. Right click 
4. Run

##### From JUnit
1. Go to TestRunner class.
2. Edit tags.
3. Right click 
4. Run as JUnit

##### Using terminal
1. Write command "mvn test"
